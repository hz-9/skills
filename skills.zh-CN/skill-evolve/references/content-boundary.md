# 内容边界标准 — 定义 SKILL.md 与 references/ 各文件之间的内容所有权边界

## Overview

定义 `SKILL.md` 与 `references/` 各文件之间的内容所有权边界。编辑或反哺时，查询此表确定内容去向。

## 文件所有权边界

| 内容领域                                             | 应写入的文件                           |
| ---------------------------------------------------- | -------------------------------------- |
| Workflow 结构/Secure 步骤/步骤格式/分支逻辑/交互范式 | `references/workflow-standard.md`      |
| 标点符号/引号/括号/全角半角                          | `references/punctuation-convention.md` |
| 文本简化规则/安全边界                                | `references/text-optimization.md`      |
| 示例格式规范/示例自洽性规则                         | `references/example-standard.md`       |
| 目录结构与 references 文件规范                       | `references/directory-structure.md`    |
| Rules 编写标准                                       | `references/rules-standard.md`         |
| Review List 编写标准                                  | `references/review-list-standard.md`   |
| 变量声明规范                                           | `references/rules-standard.md`         |
| 元数据规范/结构规范/内容规范/行为规范/防御规范/验证规范 | 保留在 SKILL.md Rules                  |
