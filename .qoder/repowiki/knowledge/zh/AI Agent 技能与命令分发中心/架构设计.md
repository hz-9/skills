- **统一分发机制**：根目录下的 `scripts/` 提供标准化的 Shell 安装脚本（`install-skills.sh`, `install-commands.sh`），支持从 GitHub 远程克隆或本地路径部署，并自动处理目标目录初始化与版本冲突。
- **多语言路由策略**：安装脚本内置交互式语言选择逻辑，根据用户偏好动态映射至 `skills`/`skills.zh-CN` 或 `commands`/`commands.zh-CN` 子模块，实现物理隔离但逻辑统一的资源加载。
- **标准化契约定义**：通过 `templates/SKILL.md` 确立所有子模块技能的结构规范（Frontmatter + 快速开始 + 工作流），确保不同语言和技能类别的元数据一致性。
- **废弃资产治理**：根目录维护 `deprecated/` 文件夹，集中管理已下线或重构的技能与命令，为子模块的演进提供清晰的迁移路径与版本回溯支持。
