- **核心架构**：采用“模板驱动 + 规范约束”的模块化设计。以 `skill-evolve/template.md` 为结构基准，通过 `workflow-standard.md` 等参考文档定义严格的编写范式。
- **分层设计**：
  - **基础层**：`skill-create` 负责从零构建符合标准目录结构的初始技能。
  - **进化层**：`skill-evolve` 与 `skill-evolve-cycle` 负责技能的持续重构、内容精简与格式对齐。
  - **工具层**：集成 Git 工作流辅助（`git-commit-helper`, `git-branch-prep`, `git-cleanup`）与 Monorepo 迁移工具（`rush-to-nx`, `changeset-gen`）。
  - **认知层**：提供 `zoom-out` 等视角切换技能，辅助理解代码全局上下文。
- **依赖方向**：所有技能均强依赖于 `skill-evolve` 定义的规范体系；Git 类技能之间存在内部调用关系（如 `git-branch-prep` 调用 `git-commit-helper`）。
