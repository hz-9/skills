# SKILL Directory Structure

```
{skill-name}/
├── SKILL.md          # (Required) Core execution instructions [reference:1]
├── scripts/          # (Optional) Executable scripts
├── references/       # (Optional) Detailed reference documents [reference:2]
├── assets/           # (Optional) Static resources such as templates, images [reference:3]
├── tests/            # (Optional) Test cases, an important part of engineering [reference:4]
└── schemas/          # (Optional) Data schemas for passing data between Skills to enable chaining [reference:5]
```
