# 测试策略决策树

选择合适测试方法的决策树。

```
所需测试类型：
│
├─ 测试业务逻辑？
│  └─ 单元测试
│     ├─ 工具：Jest、@nestjs/testing
│     ├─ Mock 所有依赖
│     ├─ 隔离测试 service 方法
│     └─ 快速执行（每个测试 < 1s）
│     │
│     └─ 使用时机：
│        ├── 测试 service 方法
│        ├── 测试验证逻辑
│        ├── 测试转换逻辑
│        └── 测试工具函数
│
├─ 测试 API 契约？
│  └─ 集成测试
│     ├─ 工具：Jest、@nestjs/testing、测试数据库
│     ├─ 使用真实数据库（测试环境）
│     ├─ 仅 Mock 外部服务
│     └─ 中等执行速度（每个测试 1-5s）
│     │
│     └─ 使用时机：
│        ├── 测试数据库查询
│        ├── 测试 repository 方法
│        ├── 测试模块间交互
│        └── 测试 guards/pipes/interceptors
│
├─ 测试用户流程？
│  └─ E2E 测试
│     ├─ 工具：Jest、Supertest
│     ├─ 完整应用引导
│     ├─ 真实数据库 + HTTP 请求
│     └─ 慢速执行（每个测试 5-30s）
│     │
│     └─ 使用时机：
│        ├── 测试完整的 API 端点
│        ├── 测试认证流程
│        ├── 测试 HTTP 级别的验证
│        └── 测试错误响应
│
├─ 测试性能？
│  └─ 负载测试
│     ├─ 工具：k6、Artillery
│     ├─ 模拟并发用户
│     ├─ 测量响应时间
│     └─ 非常慢的执行（数分钟）
│     │
│     └─ 使用时机：
│        ├── 生产部署前
│        ├── 重大变更后
│        ├── 扩展基础设施时
│        └── 识别瓶颈
│
└─ 测试安全性？
   └─ 安全测试
      ├─ 工具：OWASP ZAP、自定义中间件测试
      ├─ 测试 auth guards
      ├─ 测试输入清理
      └─ 测试速率限制
      │
      └─ 使用时机：
         ├── 实现认证后
         ├── 生产部署前
         ├── 安全审计后
         └── 定期安全审查
```

## 测试金字塔

```
        /\
       /  \      E2E 测试（10%）
      /────\    - 慢速、昂贵、全面
     /      \
    /────────\  集成测试（20%）
   /          \ - 中等成本、测试交互
  /────────────\
 /              \ 单元测试（70%）
/────────────────\ - 快速、廉价、聚焦
```

## 测试选择指南

### 单元测试（70% 的测试）
- **何时**：测试独立逻辑
- **速度**：快速（< 1s）
- **成本**：低
- **Mock**：所有依赖
- **示例**：Service 方法计算

### 集成测试（20% 的测试）
- **何时**：测试组件间交互
- **速度**：中等（1-5s）
- **成本**：中等
- **Mock**：仅外部服务
- **示例**：数据库查询、模块集成

### E2E 测试（10% 的测试）
- **何时**：测试完整的用户流程
- **速度**：慢速（5-30s）
- **成本**：高
- **Mock**：无（真实应用）
- **示例**：完整的 API 端点测试

## 快速决策指南

| 场景 | 测试类型 | 工具 |
|----------|-----------|-------|
| Service 方法逻辑 | 单元测试 | Jest |
| 数据库查询 | 集成测试 | Jest + 测试数据库 |
| API 端点 | E2E 测试 | Supertest |
| Auth guard | 集成测试 | Jest |
| Validation pipe | 集成测试 | Jest + Supertest |
| 错误处理 | E2E 测试 | Supertest |
| 性能 | 负载测试 | k6 |
| 安全性 | 安全测试 | OWASP ZAP |
